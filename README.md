# edm유학센터 캐나다 공립 컬리지 진학 AI 컨설팅 (커리어·편입)

호주 진학 안내 페이지(edm-au-ai)의 구조·디자인을 그대로 유지하고, 내용과 진단 로직만 캐나다 공립 컬리지용으로 교체한 페이지입니다.

## 주요 기능

- 커리어·이민 트랙(6문항) / 대학 편입 트랙(4문항) 진단 → 맞춤 경로 결과
- PGWP(졸업 후 취업허가) 전공 적격 여부 3색 배지: 가능 / 조건부 확인 / 불가(비학위)
- PGWP 불가 전공 선택 시 경고 카드 + 대안 3버튼 (인접 적격 전공 / 편입 트랙 전환 / 상담)
- 권역 점수제 추천 (온타리오·BC·알버타·프레리·애틀랜틱) + 컬리지 카드 2~3곳
- "2026 캐나다 유학, 무엇이 달라졌나" 정책 고지 섹션 (학업허가 캡·PGWP 전공 제한·학위 면제)
- 결과 공유 단축 링크 (#r= 코드), 질문 뒤로가기

## 데이터

- 단일 소스: `data/canada_colleges_data.json` (index.html에 인라인 동기화)
- 근거: `docs/캐나다_공립컬리지_리서치_보고서.md` · 2026 IRCC 발표 기준
- 학비(`tuition_intl`)는 검증 전 → 전 항목 "상담 시 안내" 표기 (임의 금액 기재 금지)

## 브랜드 연결 (edm)

- JS 상단 `BRAND_CONFIG` 객체에서 로고·전화·상담·카카오·가이드 링크 일괄 관리
- 타 브랜드(종로유학원 등) 적용 시 `BRAND_CONFIG`만 교체
